Notes

Application de notes à l'instar de Google Keep ou de l'application Notes d'Apple. L'application permet de créer des notes, de les modifier et de les supprimer. Les notes sont stockées sur firebase et les images, elles, sont sur firebase storage.

Elle permet également de prendre des photos et de les stocker localement sur l'appareil de l'utilisateur.

L'application est développée avec le framework Ionic, qui permet de créer des applications mobiles multiplateformes avec des technologies web (HTML, CSS, JavaScript).
